```python
torch.linspace(start, 
				end, 
				steps, 
				*, 
				out=None,
				dtype=None,
				layout=torch.strided,
				device=None,
				requires_grad=False) -> Tensor
```

Creates a one-dimensional tensor size `steps` whose values are evenly spaced from `start` to `end`, inclusive.
